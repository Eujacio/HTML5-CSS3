Trabalhando com listas

LISTA ORDENADA

Temos 3 tipos de listas, vamos começar com as listas ordenadas.
Exemplo de lista ordenada:

  1. junior

  2. leticia

  3. yasmin

  4. ana

  5. vini

Para esse tipo de lista usamos a tag "ol".
e para cada intem da lista
usamos a tag "li"

Também pode usar a tag "type" para mudar a numeração da lista para
letras maiusculas e minusculas ou algarismo romano.

Lista de definições

A lista de definições são usadas para destacar
o termo em cima, e em baixo colocar as definições desse termo.
Usamos a tag "dt" para colocarmos o termo e a tag "dd"
para colocarmos as descrições desse termo.
Por Exemplo:

fluminense
clube de futebol do rio de janeiro
barcelona
clube de futebol da catalunha

Links

Link externos

Os links esterno é usado para deixar caminhos para acessar outros sites
para inserir um link usamos a tag (a)
Por exemplo, para acessar o meu git hub basta acessar essa link: Eujacio/github

Para links esternos, usamos um uma tag dentro do "href" chamada "target=_blank"
Que serve para abrir o link em uma nova pagina, e não sobrpor a pagina atual.

Link interno

Os links internos são usados para acessar outras partes do mesmo site
Como por exemplo:
Para acessar o capitulo 02 clique aqui Capitulo 02
nesse caso usamos a tag "next" dentro do "href" para indecarmos que está br diresionando para a proxima pagina, e nas paginas secundarias usamos a tag "priv"
