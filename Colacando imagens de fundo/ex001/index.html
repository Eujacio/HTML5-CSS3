Tags do html-css

  • ! + enter = cabesalho do html-css.
  • H1 = tag para titulos.
  • p = tag para paragrafos.
  • br = tag para quebrar linha.
  • img = tag para inserir uma imagem.
  • mark = tag paga destacar palavras.
  • ctrl + shift + p = comando para invelopar o texto com alguma tag
  • small = tag para letras peqeunas.
  • ins = usado para deixar um texto em destaque.
  • del = para deixar um texto desconsiderado.
  • code = é uado para deixar as letras com o mesmo espaço lateral.
  • pre = é usado para deixar o texto do jeito que for digitado no codigo
  • abbr = é usado para mostro o significado de abreviações quando passar o
    curso em
  • cima da sigla.
    exemplo: cv

Citações

Trabalhando com listas

LISTA ORDENADA

Temos 2 tipos de listas, vamos começar com as listas ordenadas.
Exemplo de lista ordenada:

  1. junior

  2. leticia

  3. yasmin

  4. ana

  5. vini

Para esse tipo de lista usamos a tag "ol".
e para cada intem da lista
usamos a tag "li"

Também pode usar a tag "type" para mudar a numeração da lista para
letras maiusculas e minusculas ou algarismo romano.
